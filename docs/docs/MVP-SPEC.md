# Kashaaf MVP Specification

## Overview

Kashaaf MVP is built on a three-layer architecture:

- **Layer 1: Player Identity** - Core user identification and profile management
- **Layer 2: AI Training Engine** - Video analysis and performance insights
- **Layer 3: Discovery** - Content discovery and talent search capabilities

## Roles

### PLAYER
Individual athletes who upload videos, build profiles, and receive performance analysis.

### ORG
Organizations (clubs, scouts, academies) that discover talent, review analysis reports, and manage shortlists.

## In-Scope Features

### Profile Management
- User profile creation and management for PLAYER and ORG roles
- Profile customization and visibility settings

### Feed
- Content feed displaying videos and updates
- Personalized feed based on user preferences and activity

### Upload Video
- Video upload functionality for PLAYERs
- Support for multiple video formats
- Video metadata and tagging

### Analysis Report
- AI-generated performance analysis reports for uploaded videos
- Detailed insights and metrics based on Layer 2 AI Training Engine
- Report sharing and export capabilities

### Search
- Talent search functionality for ORGs
- Filtering and advanced search options
- Search by skills, position, performance metrics

### Shortlist
- ORG ability to create and manage talent shortlists
- Shortlist organization and collaboration features

## Out-of-Scope Features

- **Payments** - Payment processing, subscriptions, and billing
- **Trials/Offers** - Trial periods, offers, and contract management

---

*This MVP focuses on core identity, AI analysis, and discovery functionality. Monetization and advanced engagement features are deferred to future releases.*
